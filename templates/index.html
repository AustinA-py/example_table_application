Welcome to Customer Service Line Viewer

How to use this application:

  • Search: Use the dropdown to select either "Account Number" or "Service Address", then type your search term. The table updates automatically as you type. Partial matches work - for example, searching "123" will find "12345" and "91234".
  • View Details: Double-click any row to open a detailed view showing asset information and a map of the location.
  • Update Info: If the service line material is unknown or marked for replacement, click the "Update Info" button to submit corrected data.
